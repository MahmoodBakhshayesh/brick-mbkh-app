# mbkh_app Mason Brick

Create a full MBKH Flutter starter project with login, home, and profile features.

_Generated by [mason][1] 🧱_

## Usage

```bash
mason make mbkh_app
```

## Variables

| Var | Default | Description |
|-----|---------|-------------|
| `project_name` | `mbkh_app` | Package name (snake_case) |
| `use_phone_shell` | `true` | Center app in a 600px-wide shell on wide screens |
| `base_url` | `https://example.com` | Site origin; REST calls use `{base_url}/api/` |
| `primary_color` | `27756a` | Brand hex (no `#`); sets `AppColors.primaryColor` and `actions` |
| `use_phone_login_flow` | `false` | `false` = username/password login; `true` = phone/OTP steps |
| `enable_offline_test_login` | `true` | `test` / `test` signs in locally without an API call |

## Getting Started

### From a published / global brick

```bash
mason get
mason make mbkh_app
```

### From this repo (local path)

```bash
cd example
mason get
mason make mbkh_app -c mason_vars.json -o app --on-conflict overwrite
cd app
flutter pub get
flutter create . --platforms=android,ios,web
flutter analyze
```

See [example/README.md](example/README.md) for details.
